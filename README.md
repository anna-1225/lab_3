# Лабораторная работа №4 — Реализация алгоритма поиска подстрок с помощью регулярных выражений
## Цель работы
Изучить теоретические основы регулярных выражений и их применение для поиска и извлечения подстрок из текста. Освоить практические навыки использования библиотечных средств работы с регулярными выражениями, а также интеграцию алгоритмов поиска в графический интерфейс приложения.

## Постановка задачи
Разработать модуль поиска подстрок с использованием регулярных выражений, интегрировать его в существующее приложение (текстовый редактор) и обеспечить наглядный вывод результатов.
Варианты задач:
- Построить РВ, описывающее логин (латиница, цифры, точка, дефис, не начинается с цифры)
- Построить РВ, описывающее положительные и отрицательные целые числа и числа с плавающей точкой (разделители запятая и точка).
- Построить РВ, описывающее денежные суммы в разных валютах.
## Решение задач
### Задача 1. Логин
Описание: поиск корректных логинов в тексте. Логин должен начинаться с буквы латинского алфавита, может содержать буквы, цифры, точки и дефисы. Каждый логин находится на отдельной строке.  
#### Регулярное выражение: `^[a-zA-Z][a-zA-Z0-9.-]*$`
+ ^ - начало строки
+ [a-zA-Z] - любая латинская буква
+ [a-zA-Z0-9.-]* - любое количество латинских букв, цифр, точек, дефисов
+ $ - конец строки             
#### Примеры строк, которые должны находиться:
+ user123
+ admin
+ test25  
#### Примеры строк, которые должны находиться:
+ _test
+ 123user
+ user@name
#### Тестовые примеры

### Задача 2. Числа (целые и с плавающей точкой)
Описание: Поиск чисел в тексте: положительные и отрицательные целые числа, числа с плавающей точкой (разделители - точка или запятая), числа в экспоненциальной форме.  
#### Регулярное выражение: `[-+]?\d+(?:[.,]\d+)?(?:[eE][+-]?\d+)?`
+ [-+]? - опциональный знак + или -
+ \d+ - одна и более цифр
+ (?:[.,]\d+)? - опциональная дробная часть (точка или запятая, а затем одна и более цифр)
+ (?:[eE][+-]?\d+)? - опциональная экспоненциальная часть (e или E, опциональный знак и  одна или более цифр)
#### Примеры строк, которые должны находиться:
+ 123
+ -456
+ 3,14
+ 3.14
+ 1.23e-4
#### Примеры строк, которые не должны находиться:
+ test
+ ,
+ -
#### Тестовые примеры

### Задача 3. Денежные суммы в разных валютах
Описание: Поиск денежных сумм в тексте. Поддерживаются различные форматы: символ валюты перед суммой, сумма перед валютой, коды валют (USD, EUR, RUB и др.), символы валют ($, € и др.).
#### Регулярное выражение: `(?:USD|EUR|RUB|GBP|JPY|CNY)\s*(?:[+-]?\d+(?:[.,]\d+)?)(?:\s*(?:тыс|млн|млрд))?|(?:[$€£¥₽])\s*(?:[+-]?\d+(?:[.,]\d+)?)(?:\s*(?:тыс|млн|млрд))?|(?:[+-]?\d+(?:[.,]\d+)?)\s*(?:USD|EUR|RUB|GBP|JPY|CNY)(?:\s*(?:тыс|млн|млрд))?|(?:[+-]?\d+(?:[.,]\d+)?)\s*(?:[$€£¥₽])(?:\s*(?:тыс|млн|млрд))?`
+ (?:USD|EUR|RUB|...) - коды валют
+ (?:[$€£¥₽]) - символы валют
+ \s* - ноль и более пробелов
+ (?:[+-]?\d+(?:[.,]\d+)?) - число (целое или дробное)
+ (?:\s*(?:тыс|млн|млрд|...))? - опциональное обозначение тысяч, миллионов и т.д.
#### Примеры строк, которые должны находиться:
+ $100
+ 100 USD
+ 50 EUR
+ $99.99
#### Примеры строк, которые не должны находиться:
+ 100
+ USD
+ $
#### Тестовые примеры
